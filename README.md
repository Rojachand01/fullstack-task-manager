# Fullstack Task Manager

A simple full-stack task management app with authentication and CRUD operations.

## Tech Stack

Frontend: React  
Backend: Node.js, Express  
Database: MongoDB Atlas  
Auth: JWT

## Features

- User registration & login
- Protected task APIs
- Create, read, update, delete tasks
- Swagger API documentation
- Connected React frontend dashboard

## Project Structure

backend/ → Express API  
frontend/ → React UI

## Setup Instructions

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm start

## API Docs

http://localhost:5000/api-docs

## Deployment

Backend → Render/Railway  
Frontend → Vercel

## Scalability Notes

- Can be split into microservices (auth, tasks)
- Add Redis caching for faster reads
- Use load balancer for high traffic
