# Fullstack Task Manager

A production-ready full-stack task management application.

## Tech Stack

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Swagger API Docs

Frontend:
- React.js
- Axios
- Protected Routes

Deployment:
- Backend → Render
- Frontend → Vercel
